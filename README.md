# ONE | TECH FOUNDATION - Back-End
Desafio da trilha "Java e Orientação a Objetos": Conversor de moedas

## Visão geral da aplicação
Nesse sistema, é possível selecionar uma opção e fazer a conversão entre duas moedas.

## Tecnologias Utilizadas
- **Linguagem:** Java
- **API:** EXchangeRate-API
- **Bibliotecas:** Gson
